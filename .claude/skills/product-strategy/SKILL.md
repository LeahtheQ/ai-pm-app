## 목적

제품 비전부터 시장 포지셔닝, 경쟁 분석, 로드맵까지 전략 수립 전 과정을 구조적으로 지원한다.

## 입력 변수

- `제품명`: 분석 대상 제품 또는 서비스
- `타겟 고객`: 주요 사용자 그룹
- `현재 단계`: 아이디어 / 초기 / 성장 / 성숙
- `핵심 문제`: 해결하려는 고객 문제
- `주요 경쟁사`: 알고 있는 경쟁 제품 (선택)

## 프레임워크

### 1. 제품 비전 선언문
형식: "우리는 [타겟 고객]이 [문제 상황]에서 [핵심 가치]를 얻도록 [차별화된 방법]으로 돕는다."

### 2. 시장 규모 산정 (TAM → SAM → SOM)
| 구분 | 정의 | 산정 방법 |
|------|------|-----------|
| TAM | 전체 가용 시장 | 상향식 또는 하향식 추정 |
| SAM | 서비스 가능 시장 | TAM × 타겟 세그먼트 비율 |
| SOM | 점유 가능 시장 | SAM × 현실적 점유율 |

### 3. Porter's Five Forces
- 기존 경쟁자의 위협 (강도)
- 신규 진입자의 위협 (진입 장벽)
- 구매자의 협상력
- 공급자의 협상력
- 대체재의 위협

### 4. 포지셔닝 맵
X축과 Y축에 핵심 가치 속성 2개를 선택해 경쟁사와 함께 배치한다.
포지셔닝 선언: "우리는 [경쟁 영역]에서 [타겟]에게 [핵심 혜택]을 제공한다. [대체 솔루션]과 달리, 우리는 [차별점]을 보유하고 있다."

### 5. 전략 로드맵 (Now / Next / Later)
| 기간 | 초점 | 핵심 가설 |
|------|------|-----------|
| Now (0~3개월) | 핵심 가치 검증 | |
| Next (3~6개월) | 확장 및 성장 | |
| Later (6~12개월) | 스케일링 | |

## 출력 형식

- 각 프레임워크 결과를 마크다운 테이블 또는 섹션으로 구조화
- 현황 → 분석 → 전략 권고 → 다음 단계 순서로 작성
- 수치가 있는 경우 출처 또는 가정 명시

## 품질 체크리스트

- [ ] 제품 비전이 측정 가능한 가치를 포함하는가?
- [ ] 시장 규모 산정에 가정이 명시되었는가?
- [ ] 경쟁 분석이 단순 나열이 아닌 인사이트를 도출하는가?
- [ ] 로드맵이 가설 검증 순서로 배열되었는가?
- [ ] 다음 단계 액션이 구체적이고 즉시 실행 가능한가?
